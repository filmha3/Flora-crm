# Flora CRM

## اجرای محلی (اختیاری، برای تست)
```
npm install
npm run dev
```

## دیپلوی روی وب (برای نصب روی گوشی)
راهنمای کامل را در پیام کلود دنبال کن: آپلود به گیت‌هاب و ایمپورت در Vercel.

## نکته مهم درباره دکمه «تولید آگهی با AI»
این دکمه از یک فراخوانی مستقیم به api.anthropic.com استفاده می‌کند که فقط داخل محیط Claude.ai کار می‌کند.
بعد از دیپلوی مستقل، این دکمه کار نخواهد کرد مگر یک بک‌اند کوچک با کلید API خودتان اضافه کنید.
